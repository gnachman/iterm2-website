### iTerm2 Companion Plugin
This is an optional component you can install to pair iTerm2 with the iTerm2 Companion app on your iPhone.

It provides the only network path the companion feature uses to reach the relay. It exists as a separate component so that connecting to a companion device requires a deliberate, separate install: with no plugin present, iTerm2 cannot reach a companion device at all.

#### How to Install
Download the package and unzip it. Place in `/Applications`. You don't need to run it; iTerm2 will locate it automatically.
